# Проект по реализации приложения для анализа транзакций

## Описание
Приложение будет генерировать JSON-данные для веб-страниц, формировать Excel-отчеты, а также предоставлять другие 
сервисы.

1. Клонируйте репозиторий:
```
git clone https://github.com/username/coursework_1.git
```
2. Для работы с внешним API необходимо создать файл .env в корневой директории проекта и добавить 
в него переменные окружения: 

Для конвертации валюты воспользуйтесь Exchange Rates Data API: https://apilayer.com/marketplace/exchangerates_data-api
Для конвертации акций воспользуйтесь Twelvedata: https://twelvedata.com/
```
API_KEY=API-ключ для вывода валют
API_KEY_STONCKS=API-ключ для вывода акций
```

## Использование:

1. Откройте проект в вашей среде разработки.
2. Запустите файл main.py
```
python main.py
```
3. Добавьте новые тестовые файлы и запустите с помощью команды:

```
python -m tests.name_file
```

## Модуль `main`:
Используется для вызова основных функций.

## Модуль `open_file`:
Используется для открытия EXCEL-файла и записи данных в DataFrame и также записи данных в JSON-файл.

## Модуль `views`:
Используется для создания функций для генерации JSON-ответов. Анализирует данные одного месяца.

## Модуль `services`:
Используется для анализа, какие категории были наиболее выгодными для выбора в качестве категорий повышенного кешбэка.

## Модуль `reports`:
Используется для анализа трат по заданной категории за последние три месяца (от переданной даты).
